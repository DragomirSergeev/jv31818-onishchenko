Система управления и мониторинга Wiy

Система предназначена для установки клиентского ПО на микрокомпьютеры с конфигурацией пользователя. Позволяет подключать любые устройства по протоколам Wi-Fi, one-wire,bluetoth,ethernet и другие. Данный хост является центральной управляющей панелью для всех устройств пользователя. Реализуется REST-API, для управления с любых устройств. Для управления своими устройствами необходимо авторизоваться в системе.

Возможности системы:

    • управление любыми клиентскими устройствами через контроллеры raspberry pi с предустановленной JAVA
      Поддержка любых протоколов
      управление с любых устройств через REST (DEV)
      возможность использовать сценарии
      группировка устройств по объектам
      видеонаблюдение и триггеры на него (DEV)
